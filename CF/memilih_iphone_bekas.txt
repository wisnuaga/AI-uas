ATURAN :
R1 :
IF KEBUTUHAN STORAGE TIDAK BESAR
          CF FAKTA = -0.6
AND BUDGET 2 JUTAAN
     CF FAKTA = -1
THEN IPHONE 6 32 GB

(CF ATURAN R1 = 0.7)

JIKA OPERATORNYA AND MAKA PILIH CF FAKTA
YANG PALING KECIL, JADI CF FAKTA = -1

CF KESIMPULAN = CF FAKTA * CF ATURAN R1
R1            = -1       * 0.7
              = -0.7
(TIDAK BEGITU DIREKOMENDASIKAN
IPHONE 6 32 GB)

R2 :
IF KEBUTUHAN STORAGE BESAR
     CF FAKTA = 0.6
AND BUDGET 3 JUTAAN
     CF FAKTA = 1
THEN IPHONE 6 64 GB

(CF ATURAN R2 = 0.6)
CF FAKTA PILIH YANG KECIL KARENA OPERATOR-
NYA AND, JADI CF FAKTA = 0.6
CF KESIMPULAN = CF FAKTA * CF ATURAN R2
R2              0.6      * 0.6
                0.36
JADI CF KESIMPULAN IPHONE 6 64 GB
= 0.36 (MUNGKIN DIREKOMENDASIKAN
IPHONE 6 64GB)

R3 :
IF GARANSI RESMI
       0.6
AND (MASIH GARANSI
        -1
OR GARANSINYA BELUM LAMA HABIS)
     0.6
THEN BERKUALITAS

(CF ATURAN R3 = 0.9)
JIKA OPERATORNYA OR MAKA DIPILIH CF YANG
PALING BESAR, CF UNTUK MASA GARANSI = 0.6
CF FAKTA UNTUK GARANSI RESMI 0.6
CF FAKTA UNTUK MASA GARANSI (HASIL DARI
OR = 0.6)
CF FAKTA KESELURUHAN = 0.6

CF KESIMPULAN R3 = CF FAKTA * CF ATURAN R3
(BERKUALITAS)      0.6      * 0.9
                   0.54
JADI CF UNTUK BERKUALITAS ADALAH 0.54
(KEMUNGKINAN BERKUALITAS)

R4 :
IF JARINGAN SELULER NORMAL (CF=1)
AND TOMBOL HOME EMPUK (CF=0.6)
AND TOMBOL POWER BERFUNGSI (CF=1)
AND EARPHONE ORI (CF = 0.05)
AND CHARGER NORMAL (CF = 1 )
AND JACK AUDIO LANCAR (CF = 1 )
AND SPEAKER NORMAL (CF = 1)
THEN BERKUALITAS

(CF ATURAN R4 = 0.8)
CF FAKTA = KARENA OPERATOR AND MAKA PILIH
YANG PALING KECIL YAITU 0.05
CF KESIMPULAN R4 = CF FAKTA * CF ATURAN R4
(BERKUALITAS).     0.05     * 0.8
                   0.04

JADI CF BERKUALITAS = 0.04
(KUALITAS SEDANG)

TAHAPAN CERTAINTY FACTOR (CF) :
1. MENYUSUN ATURAN / RULES (DIDAPAT DARI
PAKAR, DISUSUN DEVELOPER)
2. TENTUKAN CF DARI ATURAN / CF ATURAN
(DIDAPAT DARI PAKAR, DIBERI NILAI OLEH
DEVELOPER)
3. TENTUKAN CF DARI FAKTA (DIDAPAT DARI
USER, DIBERI NILAI OLEH DEVELOPER)
4. TENTUKAN CF DARI KESIMPULAN
   (DIDAPAT DARI HASIL PERHITUNGAN)
